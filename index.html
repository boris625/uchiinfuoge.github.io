1 Задание №1: Единицы измерения информации

Бит → Байт → Килобайт → Мегабайт → Гигабайт *8 ← /8 *1024 ← /1024 *1024 ← /1024 *1024 ← /1024 Формулы: N = 2^i I = K * i

Пример из учебника: В некоторой кодировке 1 символ кодируется 32 битами. На доске написан текст: «Борнео, Суматра, Хонсю, Виктория, Мадагаскар, Калимантан – острова». Ученик вычеркнул из списка название одного из островов. Заодно он вычеркнул ставшие лишними запятые и пробелы. При этом размер нового предложения в данной кодировке оказался на 32 байта меньше, чем размер исходного предложения. Напишите в ответе вычеркнутое название острова.

Решение:

  1. I = 32 байта (объем уменьшился на 32 байта)
  2. i = 32 бит = 32:8 = 4 байта (информационный вес одного символа)
  3. K = I / i = 32 байта / 4 байта = 8 символов (количество удаленных символов)
  4. Из 8 символов были удалены: 1 запятая, 1 пробел и 6 букв
  5. Находим слово длиной 6 символов - это слово "Борнео"

Ответ: Борнео

8 Задание №8: Поисковые запросы

Задача: В таблице приведены запросы и количество найденных по ним страниц некоторого сегмента сети Интернет.

Запрос Найдено страниц (в тысячах)
Хоккей & Россия 168
Хоккей & Канада 237
Хоккей & Россия & Канада 79

Какое количество страниц (в тысячах) будет найдено по запросу Хоккей & (Россия | Канада)?

Решение (круги Эйлера):

Обозначим области:

  • Область 1: Хоккей & Россия (без Канады)
  • Область 2: Хоккей & Россия & Канада (пересечение)
  • Область 3: Хоккей & Канада (без России)

Из условия:

  • Хоккей & Россия = 1+2 = 168
  • Хоккей & Канада = 2+3 = 237
  • Хоккей & Россия & Канада = 2 = 79

Находим:

  • 1 = 168 - 79 = 89
  • 3 = 237 - 79 = 158

Хоккей & (Россия | Канада) = 1+2+3 = 89 + 79 + 158 = 326

Ответ: 326

9 Задание №9: Подсчет путей в графе

Задача: На рисунке изображена схема соединений, связывающих пункты A, B, C, D, E, F, G, H. По каждому соединению можно двигаться только в одном направлении, указанном стрелкой. Сколько существует различных путей из пункта C в пункт H, проходящих через вершину E?

Решение (метод динамического программирования):

Присваиваем каждому пункту количество путей из C:

  • C = 1 (начальная точка)
  • A = C = 1
  • F = C = 1
  • B = A + C = 1 + 1 = 2
  • D = A + B = 1 + 2 = 3
  • E = F + C + D = 1 + 1 + 3 = 5
  • H = E = 5

Ответ: 5

Решение с помощью SQL-подобных запросов:

Основные SQL команды:

SELECT - выборка данных FROM - указание таблицы WHERE - условие выборки ORDER BY - сортировка GROUP BY - группировка

Примеры запросов:

-- Подсчет мужчин, сдававших информатику SELECT COUNT(*) FROM таблица WHERE пол = 'муж' AND предмет = 'информатика' -- Средний балл по математике SELECT AVG(баллы) FROM таблица WHERE предмет = 'математика' -- Количество участников по предметам SELECT предмет, COUNT(*) FROM таблица GROUP BY предмет

Решение в Excel:

  • Для подсчета: =СЧЁТЕСЛИМН(B2:B1001;"муж";C2:C1001;"информатика")
  • Для среднего: =СРЗНАЧЕСЛИ(C2:C1001;"математика";D2:D1001)
  • Для диаграммы: создать сводную таблицу и построить круговую диаграмму

Ответы из учебника:

  • Мужчин по информатике: 171
  • Средний балл по математике: 47,21
  • Соотношение по предметам: математика:323, информатика:335, физика:342
